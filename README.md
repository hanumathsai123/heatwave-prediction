# 🌡️ HeatSense - Heat Wave Prediction System

ML-powered web application to predict heat waves based on weather parameters.

## Features
- ✅ Real-time heat wave prediction
- ✅ Risk level assessment (LOW, MEDIUM, HIGH)
- ✅ Safety recommendations
- ✅ Responsive UI
- ✅ Deployed on Vercel

## Tech Stack
- Python + Flask
- Machine Learning (scikit-learn)
- HTML5/CSS3/JavaScript
- Vercel Serverless

## Quick Start
```bash
python heatwave_prediction/app.py
```

Open: http://localhost:5000
